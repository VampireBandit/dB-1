# Features


-Bluetooh connectivity 


-Wifi integratad


-Long battery life with 2380mah Battery


-Argonomic design


-Rotary encoded volum controll


-4 button navigation


-2.8"   didplay


-Cherry mx switches for fidgety feel


-Custom designed UI


-Unlimited storage(depends on your SD card)


-Programmabe environment
